# 🚀 Task Management System

A professional Task Management application built with **Next.js**, **Prisma**, and **MongoDB**.

## ✨ Features
- ✅ Add new tasks easily.
- 🗑️ Delete tasks when done.
- ✔️ Mark tasks as completed with a toggle.
- 🕒 Real-time updates and persistence with MongoDB.
- 🎨 Beautiful UI with Tailwind CSS and Lucide Icons.

## 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), Tailwind CSS
- **Backend:** Next.js Server Actions
- **Database:** MongoDB Atlas
- **ORM:** Prisma

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/Erandi-idu/task-management-system`
2. Install dependencies: `npm install`
3. Set up `.env` with your `DATABASE_URL`.
4. Run: `npx prisma db push`
5. Start dev server: `npm run dev`
